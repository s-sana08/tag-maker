# 🛠️ Tag Maker

## 📌 Overview

**Tag Maker** is a simple web-based tool that helps generate HTML code quickly and easily. It allows users to convert plain text into structured HTML elements without writing code manually.

---

## 🌐 Live Demo

https://s-sana08.github.io/tag-maker/

---

## ✨ Features

* 🔹 Generate `<ul>` and `<ol>` lists
* 🔹 Create `<option>` tags
* 🔹 Generate `<a>` (URL) tags
* 🔹 Sort text alphabetically
* 🔹 Convert special characters to HTML entities
* 🔹 Copy output with one click

---

## 🚀 How to Use

1. Clone the repository:

   ```bash
   git clone https://github.com/s-sana08/tag-maker.git
   ```
2. Open:

   ```
   index.html
   ```
3. Enter your input and click **Convert**
4. Copy the generated output
